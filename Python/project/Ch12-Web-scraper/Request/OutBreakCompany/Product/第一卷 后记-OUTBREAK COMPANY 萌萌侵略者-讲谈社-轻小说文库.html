    大家好，我是轻小说家Y一郎。

    身为一名到处写书而毫无操守可言的轻小说家的我，这回首次在讲谈社轻小说文库出书了――呃，既然本书是该文库的创刊作品之一，这不就是废话嘛（笑）。

    好啦，来谈谈这本《OUTBREAKCOMPANY萌萌侵略者》吧。其实，这故事的基本骨干本身，是我很久之前就在构想的东西。

    当初是思考要推出什么奇幻作品的企划――然而毕竟笔者在某些想法上有些乖僻，所以在想到「剑与魔法的冒险故事」之前，会在习惯上先思考「什么东西出现在奇幻世界中会变得很奇怪呢？」之类的问题。

    拿我本身的例子来说，一开始出道的作品就是从「龙在奇幻作品当中感觉就像是什么最终大魔王一样，可是如果到了现代，被军队的火箭炮集中攻击的话就没戏唱啦。」之类的想法开始，最后变成「就来想一个活在近现代的龙的故事吧。」这样啊。

    在同样的意义上，我思考着「什么东西出现在奇幻世界中会很奇怪」，而在百般思索之下，脑海中便浮现了「公司」这样的东西。

    在精灵啦、矮人啦、龙啦、蜥蜴人啦……这一东西存在的社会中，如果出现了「股份有限公司」，应该会很奇怪吧？而且如果那家公司是专卖漫画书之类的东西，应该会更怪吧？就是在这样连锁性的灵感之下，这个故事便诞生了。

    无论是现实中的世界，还是历史上的事件，只要本来不应该会相遇的东西互相接触的时候，总是会产生各式各样的「化学反应」，而且多半都会在超乎预想的形式下发生变化。如果将我们世界的娱乐产业带到异世界的话，究竟会发生什么事情――想必应该会变得一团混乱吧？不过，笔者还是希望各位读者们，能够跟着主角们一起看看这场化学变化后续的发展呢。

    2011年10月27日　Y一郎
